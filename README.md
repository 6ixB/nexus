## Nexus
An Identity Provider Platform integrated with an OAuth 2.0 and OpenID Connect Authorization Server, built using Nest.js and Node OIDC Provider.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev -- -b swc

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
